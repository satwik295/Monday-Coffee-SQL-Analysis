# ☕ Monday Coffee SQL Analysis

## About the Project

This project analyzes sales data for **Monday Coffee** using SQL. The objective was to answer business questions and uncover insights that can help improve sales performance, understand customer behavior, and identify opportunities for business growth.

By working with the data, I explored trends across different cities, products, and customers to provide meaningful, data-driven insights.

---

## What I Analyzed

Some of the questions answered in this project include:

- Which cities generate the highest revenue?
- Which coffee products are the best sellers?
- How much does each customer spend on average?
- Which cities have the largest customer base?
- Where is the biggest market opportunity?
- How are sales changing month over month?
- Which cities are the best candidates for opening new coffee stores?

---

## SQL Skills Used

This project demonstrates the use of:

- SELECT statements
- JOINs
- GROUP BY
- Aggregate functions
- Common Table Expressions (CTEs)
- Window functions
- Ranking functions
- Date functions
- Subqueries

---

## Dataset

The analysis is based on four tables:

- **city** – City information
- **customers** – Customer details
- **products** – Coffee products
- **sales** – Sales transactions

---

## Key Insights

- Identified the highest revenue-generating cities.
- Compared customer spending across different locations.
- Ranked the top selling products in each city.
- Analyzed monthly sales trends and growth.
- Evaluated potential locations for future store expansion using revenue, customer base, and operating costs.

Working on this project strengthened my SQL skills and helped me gain hands-on experience with solving real business problems using data. It also improved my understanding of CTEs, window functions, aggregations, and writing efficient analytical queries.
